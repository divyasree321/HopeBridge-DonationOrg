# HopeBridge - Nonprofit Donation Platform

HopeBridge is a premium nonprofit donation platform designed for high-impact social storytelling and seamless giving experiences. Built with a focus on trust, transparency, and emotional resonance.

## Tech Stack
- **React (Vite)**
- **Tailwind CSS (V4)**
- **Framer Motion** (Animations)
- **Lucide React** (Icons)
- **Shadcn/UI** (Components)
- **Recharts** (Impact Analytics)
- **React Router** (Navigation)

## Features
- **Dynamic Programs Discovery**: Filter and search through vetted global initiatives.
- **Conversion-Optimized Donation Flow**: Minimalist, multi-step donation UI with real-time impact calculations.
- **Radical Transparency**: Impact dashboard with interactive charts and audited statistics.
- **Story-First UX**: Deep program detail pages focused on the human element.
- **Premium Design System**: Sophisticated emerald/teal color palette with glassmorphism and smooth motion.

## Deployment
This app is ready for deployment on Vercel, Netlify, or Cloud Run.
To deploy to Vercel:
1. `npm run build`
2. Connect repository to Vercel dashboard.

## Hackathon Submission Notes
HopeBridge aims to bridge the "trust gap" in philanthropy. By combining award-winning UI/UX with verified data and emotional storytelling, we increase donor conversion rates and promote long-term monthly support (Bridge Builders).
